细分着色器02——Loop Subdivision

2021-09-12 图形学 OpenGL 0 评论

Loop Subdivision

Loop Subdivision 通常用于三角形网格。

实现的简单步骤如下：

  • 将每个三角形拆成四个三角形，并计算每个新的顶点的坐标。
  • 重新计算旧的顶点的坐标。

新顶点计算


GAMES101 Lecture12 P13

细分创建的点受到其接壤的两个三角形所有顶点的影响。

旧顶点计算


GAMES101 Lecture12 P14

旧顶点的新坐标计算即受到自身原坐标的影响，也受到附近旧顶点坐标的影响。

伪代码

代码实现的基本思想：使用旧的 VBO 和 EBO 反序列化成细分结构化数据，细分计算后，再序列化成新的 VBO 和 EBO 。

序列化操作比较简单但繁琐，其实也是遍历每个 vertex object，然后根据它存储的周围顶点构成三角形图元的索引值。去掉重复的，最后得到新的 VBO 和 EBO。

总结

以上实现都是在 CPU 上实现 Loop Subdivision，在显卡是 MX150 的小米笔记本上，细分度为 4、顶点总数为 2562 的操作耗时 8 秒，细分度为5、顶点总数为 10242 的操作耗时 221 秒。但博主觉得这耗时数据应该没什么参考价值，虽然代码还有优化的空间。
